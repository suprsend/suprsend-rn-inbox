# @suprsend/rn-inbox

SuprSend SDK for integrating inbox functionality in ReactNative applications

## Installation

```sh
npm install @suprsend/rn-inbox
```

## Usage

```js
import { multiply } from "@suprsend/rn-inbox";

// ...

const result = await multiply(3, 7);
```
